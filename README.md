# Support Intent Router

AI-oriented TypeScript service that classifies customer support messages, extracts urgency signals, and routes tickets to the right queue.

## Highlights

- Node.js + TypeScript API service
- Interpretable intent scoring
- Queue routing logic with confidence explanations
- Unit tests for classification behavior

## Run locally

```powershell
npm install
npm run dev
npm test
```

## Tech stack

Node.js, Express, TypeScript, Vitest.
